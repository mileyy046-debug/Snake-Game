# Snake Game

## Integrante
Miley Yugsi 

## Objetivo del sistema
Desarrollar un videojuego Snake utilizando Python, aplicando estructuras lógicas, programación y control de versiones con GitHub.

## Descripción de funcionalidades
- Controlar la serpiente con las teclas de dirección.
- Comer la comida para aumentar el tamaño de la serpiente.
- Incrementar la puntuación al comer.
- Finalizar el juego cuando la serpiente choque con una pared o consigo misma.

## Tecnologías utilizadas
- Python
- GitHub
